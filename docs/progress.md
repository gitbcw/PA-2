  - 消息模型
  - 分类和标签模型
- 实现 API 路由
  - 用户 CRUD 操作

- 支持OpenAI API
- 支持Anthropic API

- 支持自定义LLM API
  - 可连接到任何兼容的LLM服务
  - 支持流式输出
  - 支持自定义请求头和参数
